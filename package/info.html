Clawd Monitor

CODEX // CLAWD [LIVE]

在 HoloCubic 上显示 Codex 实时工作状态、会话信息和未来 8 小时天气。

核心功能

实时状态10 类 Codex Hook 事件与独立 Clawd 动画
天气仪表逐小时降雨、温湿度、风向、风速与阵风
三页显示左右键切换状态、天气和会话页面
双语字体中英文自动匹配字体，并使用 RGB 次像素渲染
设备管理通过 /holo_pet/ 配置桥接主机并测试连接

界面预览

部署

  1. 将 package/ 内全部文件复制到 /sd/apps/holo_pet/ 或商店分配的 /sd/apps/holopet/，然后重扫 app。
  2. 在 Codex 电脑运行 node bridge/install-codex-hook.js。
  3. 在管理页填写电脑局域网 IP；默认端口 17321，路径 /events。
